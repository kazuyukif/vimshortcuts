Wim shortcuts
——————————————
w    : move forward one Word
b    : Backfoward one word
$    : End of the line
0    : Begining of the line
gg   : move to TOP
G    : move to END

-----------------------------
u     : Undo the last command
^r    : Re-undo (cancel undo)

d$    : Delete all till end of the line
d0    : Delete all till BEGINING of the line
dw    : Delete one word

dd    : Delete one line
<n>dd : Delete <n> lines from the current line below
yy    : Yank one line
<n>yy : Yank <n> lines
p     : Paste yanked buffer

-----------------------------
/<keyword>    : search for <keyword>
n             : Next search
N             : Previous search word

:%s/<keyword>/<replace>/        : Search and replace NO CONFIRMATION (ALL)
:%s/<keyword>/<replace>/g       : Search and replace NO CONFIRMATION (ALL)
:%s/<keyword>/<replace>/gc      : Search and replace with confirmation

-----------------------------
:set number   : set line numbers

Cmd shortcuts
——————————————
^a		: Start of line
^e		: End of line
Option-<	: move one word back
Option-> : move one word forward

^l		: Clear screen

^w		: Delete one word
^y		: Cancel one word

^k		: Delete all after the cursor
^u		: Delete all before the cursor

Chrome shortcuts
——————————————
^TAB	: Next TAB
*TAB	: Select Window
*w		: Close current TAB
*+/-		: Change font size

cVim shortcuts
——————————————
?	: Help
H	:  go back in History
L	: go forward in history

t	: new Tab
J	: Go one tab left
K	: Go one tab right
x	: close current tab

yy 	: copy current URL to clipboard
p      : Open the clipboard’s URL
